# Pokedex App

A React Pokedex application that fetches Pokemon data from the PokéAPI.

## Features

- Browse 20 Pokemon at a time
- Click on a Pokemon to see details (type, stats, abilities, height, weight)
- Navigate between pages with Next/Previous buttons
- Navigation with multiple pages

## Setup

1. Install dependencies:
   ```
   npm.cmd install
   ```

2. Run development server:
   ```
   npm.cmd run dev
   ```

3. Open the localhost link shown in terminal

## Build

To build for production:
```
npm.cmd run build
```
